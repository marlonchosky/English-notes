## 1. Accusative Verbs (Transitive Verbs)

An **Accusative Verb** is the formal syntactic term for a traditional **Transitive Verb**. It is named "accusative" because, in morphologically case-marked languages (like Latin, German, or Old English), the direct object is assigned the **accusative case**.

* **Core Logic:** `Subject (Agent/Doer) → Verb → Object (Patient/Undergoer)`
* **Example:** *The chef **broke** the egg.*
    * **The chef:** The active agent initiating the action.
    * **the egg:** The direct object receiving the action (structurally assigned accusative case).

---

## 2. Unaccusative Verbs (Patient-Subject Intransitives)

An **Unaccusative Verb** is an intransitive verb where the grammatical subject does not cause the action, but instead acts as the **Patient** or **Theme** undergoing a spontaneous change of state. 

* **The Core Logic:** `Subject (Patient) → Verb`. Something happens *to* the subject itself. Because there is no object slot to receive accusative case, the verb is termed **un-accusative**.
* **Example:** *The egg **broke**.*
    * **The egg:** It sits in the subject slot, but semantically remains the entity affected by the change of state.

---

## 3. Structural Interaction with Resultatives

Because the subject of an unaccusative verb is an underlying patient, these structures naturally welcome **Resultative Secondary Predicates** to describe the final state of the subject. These split into two syntactic categories:

### A. Obligatory Resultative Complements
With certain unaccusative verbs of motion or weak state changes, the sentence feels empty, vague, or grammatically incomplete without a resultative to establish a definitive endpoint.
* *The door **blew** **open**.*
    * **Analysis:** If you state *"The door blew,"* the action lacks an endpoint boundary. The adjective *open* serves as an obligatory **Resultative Complement**.

### B. Optional Resultative Adjuncts
For robust unaccusative verbs that inherently contain a full semantic event, a resultative adjective can be added optionally to emphasize the final, absolute state.
* *The river **froze** **solid**.*
    * **Primary Predicate:** *The river froze.* (A complete, grammatical sentence on its own).
    * **Secondary Predicate:** *The river became solid.*
    * **Syntactic Role:** *Solid* is a **Resultative Adjunct** (not a Subject Complement, as *froze* is a true unaccusative verb, not a copular linking verb).

### C. Prepositional Resultatives (The Transition Endpoint)
When an unaccusative verb denotes a radical physical transformation, division, or fragmentation, the result must be mapped via a Prepositional Phrase.
* *The vase **broke** **into pieces**.*
    * **Syntactic Role:** *into pieces* functions as a **Prepositional Resultative Complement**. It provides the obligatory bounded path showing the exact state transition of the subject.
